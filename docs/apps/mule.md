# Mule

**P2P File Sharing Client**

Mule is a desktop P2P file sharing application designed for the decentralized web. It combines BitTorrent capabilities with Shogun's GunDB network for metadata and searching.

## Features

- **Torrent Client**: Download and seed files via BitTorrent protocol.
- **Network Search**: Search for content across the Shogun network (Internet Archive integration via Relay).
- **Cross-Platform**: Available for Windows, macOS, and Linux.
- **Streaming**: Watch videos and listen to audio while downloading.
